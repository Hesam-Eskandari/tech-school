# C++

## Prerequisites

### Keywords
- Reserved words that make the language grammar
- There are more than 90 keywords in C++
- Stream insertion operator: `<<`
- Stream extraction operator: `>>`
- Scope resolution operator: `::`
- Example:
```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```
